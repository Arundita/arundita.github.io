Timber Guy using C++ with SFML

Introduction
This is a project to crete the Timber Guy game using C++ with SFML

Working Theory
1. Created Shapes and added movement to the shapes (clouds, tree, buzzing bee, the palyer, axe, and log) using SFML Texture, Sprite and draw.
2. Added functionality to pause and restart the game.
3. Added score and message text using the Text and Font classes of SFML.
4. Added a timebar, added the functionality for growing branches.
5. Added sounds using SFML sound class.
6. Handled player's input (keyboard's input, mouse positions and inputs) and player's death(removing/replacing shapes).
